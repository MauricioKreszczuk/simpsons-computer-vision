# Datos del Proyecto

Debido al peso de las imágenes originales (más de 42,000 capturas), el dataset no se encuentra alojado en este repositorio. 

Para replicar el proyecto, debés descargar los datos originales desde Kaggle:
* **Dataset:** [The Simpsons Characters Dataset (Alex Attia)](https://www.kaggle.com/datasets/alexattia/the-simpsons-characters-dataset)

Asegurate de colocar la carpeta `simpsons_dataset` aqui.
